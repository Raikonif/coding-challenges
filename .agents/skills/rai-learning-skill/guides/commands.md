# Commands Guide

Commands may be invoked as plain commands (`start`) or slash-style aliases (`/start`).

- `start`: begin session for topic.
- `next`: advance or resume according to current state.
- `pause`: pause current step and wait for learner response.
- `resume`: continue from paused state.
- `skip`: skip current subtopic and move forward.
- `restart`: restart from subtopic 1 with current setup.
- `stop`: end session with summary and next steps.
- `quit`: end immediately without summary.
- `level easy|medium|hard`: adjust difficulty (`easy->basic`, `medium->intermediate`, `hard->hard`).
