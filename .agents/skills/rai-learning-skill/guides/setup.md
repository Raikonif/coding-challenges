# Setup Guide

Collect missing setup values before teaching:
1. `level`
2. `mode`
3. `explanation`
4. `active_creation_path`

Ask only for missing fields.

## Conservative Value Mapping
- `beginner/easy/new` -> `basic`
- `medium/average/some experience` -> `intermediate`
- `advanced/expert/difficult` -> `hard`
- `learn/teach me/guide me` -> `student`
- `quiz me/interview/challenge me` -> `interview`
- `short/quick/brief` -> `concise`
- `standard/balanced` -> `normal`
- `detailed/thorough/complete` -> `deep`
- `build it/runnable/hands-on/create files` -> `yes`
- `explain only/theory/chat only/no files` -> `no`

## Configuration Semantics
- `basic`: slower pace, more definitions, smaller steps.
- `intermediate`: balanced depth and pace.
- `hard`: advanced tradeoffs, edge cases, stronger validation.
- `student`: explain first, then check.
- `interview`: ask first, evaluate, then teach gaps.
- `concise`: compact explanation.
- `normal`: one explanation + one example + one checkpoint.
- `deep`: layered explanation, multiple examples/checks.
